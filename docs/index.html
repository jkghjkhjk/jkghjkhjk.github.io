早期生涯

西田有志誕生在日本縣三重市,從小就喜歡排球，第一次打排球是在幼兒園，因為哥哥和姐姐們都打排球。

西田有志的家鄉位於名古屋西的三重縣員弁市，西田有志中學時代表三重縣參加了日本縣級初中排球賽，原本被期 望加入排球傳統高中星城高中，不過西田有志最 終留在了三重縣四日市市海星高中。

西田有志2017年入選日本U19國少隊，以替補身份參加了2017年U19亞少賽。在2018年高中畢業前，西田有志加入了日本V聯賽JTEKT Stings隊。2018年1月6日，西田有志上演處 子秀，以替補身份出場得到10分，全部來自扣球。

運動生涯

2018年，18歲的西田有志憑藉僅1米86的身高，先後參加了國家聯賽和世錦賽,現今在義大利排球聯賽並效力於Volley Callipo俱樂部。
